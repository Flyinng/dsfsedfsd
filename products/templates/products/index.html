Sobre nós

Nós da Robin, somos uma equipe que formada no projeto da Mostra de Talentos, do Programa Entra21 na Turma de Python, com esse projeto buscamos auxiliar as pessoas a obterem o melhor preço e os melhores produtos na hora de montar seu computador.

Equipe: Vinícius Reisch, Ketlin Kelli Ciquelero, Daniel Ferrera, Gabriel Eduardo Ritter

Sobre o projeto

Robin é um website que visa ajudar as pessoas na hora de escolher as peças para montar seu computador, o Robin utiliza de Automação e Web-Scraping em Python para reunir dados de sites de informática, reúne e filtra esses dados em um Banco de Dados MySQL online, via AWS, depois utilizando o Framework Django, os dados são reunidos em website responsivo.
